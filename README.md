# LinLED Coffee Machine UI

### Prerequesites

- Python >= 3.10

### Instalation

#### Windows

```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```

### Build

```bash
pyinstaller --onefile linled_coffee.py.py
```

### Try on local

```bash
python linled_coffee.py.py
```
